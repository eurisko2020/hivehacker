Update via USB Stick

No computer needed — update from a USB stick:

  1. Download the .raucb file from hivehacker.ca
  2. Copy it to a FAT32 USB stick in a folder called hivemapper_update/
  3. Power OFF this dashcam
  4. Insert the USB stick
  5. Power ON — update installs automatically in 2-3 minutes
